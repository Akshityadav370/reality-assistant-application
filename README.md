# Food Ordering Application
 
This application allows users to order food by scanning a QR code. After scanning, users are redirected to a menu page where they can select items and send the order via WhatsApp to kitchen.
 
## Features
 
- Scan QR code to access the menu
- Browse and select food items
- Send the order via WhatsApp
 
## Getting Started
 
Follow these instructions to set up and run the project on your local machine.
 
### Prerequisites
 
- Node.js (v14.x or later)
- npm (v6.x or later)
 
### Installation
 
1. Clone the repository:
 
    ```
    git clone "repo name"
    cd food-ordering-app
    ```
 
2. Install the dependencies:
 
    ```
    npm install
    ```
 
### Building the Project
 
To build the project, run the following command:
 

npm run build

### To run the Project
npm run 
npm start
